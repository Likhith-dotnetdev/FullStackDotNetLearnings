I think you have an OrderService and a PaymentService. The requirement is: OrderService should call PaymentService using Dependency Injection. How do you implement this.

Ex:

1.Interface:

public interface IPaymentService
{
    bool ProcessPayment(decimal amount);
}

2.impl payment service :

public class PaymentService : IPaymentService
{
    public bool ProcessPayment(decimal amount)
    {
        Console.WriteLine($"Payment of ₹{amount} processed.");
        return true;
    }
}

3.inject into order service via DI

public class OrderService
{
    private readonly IPaymentService _paymentService;
 OrderService(IPaymentService paymentService)
    {
        _paymentService = paymentService;
    }

    public void PlaceOrder(decimal amount)
    {
        Console.WriteLine("Order placed.");
        _paymentService.ProcessPayment(amount); // calls PaymentService
    }
}


4. Register here

builder.Services.AddScoped<IPaymentService, PaymentService>();
builder.Services.AddScoped<OrderService>();

 How do you find a Duplicates from student Raw table without having any indexes and keys?
Use group by, rownumber with partitionby, rank, dense_rank
WITH CTE AS (
    SELECT *,
           ROW_NUMBER() OVER (PARTITION BY Name, Salary ORDER BY Id) AS rn
    FROM Employee
)
DELETE FROM CTE
WHERE rn > 1;
